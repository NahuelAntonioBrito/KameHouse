# Carrito de Compras con React

Bienvenidos a mi tienda KameHouse, es un e-comerce, en este proyecto he utilizado React, React-Boostrap. Firebase, React-Router-Dom y le di estilos con css. 

## Descripción

El objetivo de este proyecto es poder realizar un e-comerce orientado a la venta de productos geek como pueden ser Comics, Mangas, Animes en blu ray o Figuras de colección. La idea es crear una página funcional y amigable con el usuario, de simple entendimiento. Utilizando tecnologías como React y otras herramientas que mejoran la experiencia del usuario.

### Características

-	Interface de usuario amigable.
-	Un listado de productos con la posibilidad de ver productos por  categorías.   
-	Agregar productos al carrito.
-	Actualización de la cantidad de elementos del carrito .
-	Generar ordenes de compras.
-	Almacenar la orden de compra en Firebase.


### Tecnologías utilizadas

- React: Biblioteca de JavaScript paracontruir interfaces de usuario interactivas.
-React-Boostrap: Conjunto de componentes de interface de usuario.
-React-Router-Dom: Enrutador de enlaces para React.
-Creaqate-React-App: Herramienta de línea de comandos que configuran automaticamente un proyecto de React.
